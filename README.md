# movielens_analysis

В проекте проанализрована короткая версия одного из датасетов с MovieLens на 1000 строк.

Цель проекта – выявить статистику и обогатить данные через парсинг. Реализованы 5 классов: 4 класса для обработки 4х CSV файлов и 1 класс для проведения тестирования. 

Проект состоит из 2х файлов: реализации классов и отчет по проведенный работе, в котором проводится тестирование работы классов.

Описание csv файлов:
movies.csv – содержит основную ифнормацию о фильмах: идентификатор фильма, заголовок, список жанров. 
ratings.csv – содержит оценки пользователей на фильмы, id пользователя, идентификатор фильма, время выставления оценки. tags.csv – содержит пользовательские теги (теги определяет пользователь, нет конкретного списка), id пользователя, id фильма.
links.csv – содержит внутренний id фильма, imdid - идентификатор фильма на imdb, tmdbid - идентификатор фильма на tmbd.
